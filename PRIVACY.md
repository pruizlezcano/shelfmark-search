# Privacy Policy

> Last updated: April 1, 2026

Shelfmark search does not collect any data (personal, behavioural, etc) whatsoever.

Chrome's local storage is used to save you Shelfmark URL as saved sessions.

All the code is Open Source and publicly viewable at:

- https://github.com/pruizlezcano/shelfmark-search

If you have any questions, raise an issue at:

- https://github.com/pruizlezcano/shelfmark-search/issues
